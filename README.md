# 🧹 Data Cleaning Project (Data Preprocessing Practice)

## 📌 Overview

This project focuses on data cleaning and preprocessing as part of my early AI ML bootcamp journey. The main goal is to prepare raw data into a clean and usable format for further analysis or modeling.

## 🎯 Objectives

* Identify and handle missing values
* Detect and treat duplicate data
* Clean inconsistent or invalid data entries
* Prepare dataset for analysis or machine learning

## 📂 Dataset

The dataset used in this project contains raw data that requires preprocessing before analysis.
It includes potential issues such as:

* Missing values
* Duplicate records
* Inconsistent formatting

## 🛠 Tools & Libraries

* Python
* Pandas
* NumPy
* Google Colab

## 🔍 Key Tasks Performed

* Handling missing values (drop / fill strategies)
* Removing duplicate data
* Data type correction
* Basic data validation and consistency checks

## 📊 Workflow

1. Data Understanding
2. Data Cleaning

   * Handling missing values
   * Removing duplicates
   * Fixing data types
3. Data Validation
4. Final Clean Dataset

## ▶️ How to Run

1. Clone this repository:

   ```
   git clone https://github.com/phoeb-exe/california-housing-data-cleaning.git
   ```
